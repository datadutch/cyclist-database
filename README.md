# Cyclist Database

A database for cyclist information.

## Description

This repository contains a database for storing and managing cyclist information.

## Features

- Store cyclist details
- Manage cyclist records
- Query cyclist data

## Getting Started

### Prerequisites

- Python 3.x
- SQLite (or your preferred database)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/datadutch/cyclist-database.git
   ```

2. Navigate to the project directory:

   ```bash
   cd cyclist-database
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

### Usage

Run the application:

```bash
python main.py
```
